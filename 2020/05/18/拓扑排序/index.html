拓扑排序

发表于 2020-05-18 | 分类于 数据结构与算法

对一个有向无环图(Directed Acyclic Graph简称DAG)G进行拓扑排序，是将G中所有顶点排成一个线性序列，使得图中任意一对顶点u和v，若边∈E(G)，则u在线性序列中出现在v之前。通常，这样的线性序列称为满足拓扑次序(Topological Order)的序列，简称拓扑序列。简单的说，由某个集合上的一个偏序得到该集合上的一个全序，这个操作称之为拓扑排序。

举个例子

学校选课，某些课都要求先修课，比如说，我们学校的《Java语言程序设计》课程就要求预修课程《面向对象——C++》，（PS.尽管没什么必要)，然而然而这就构成了一个java课指向c++课的有向无环图，这个学习顺序也就构成了一个拓扑序列 [java,c++]。

比如说这张图的一个满足条件的拓扑排序就是[1，2，4，3，5]。

dfs
bfs

bfs更容易理解，建立入度表，并把入度为0的节点放入队列，依次出队列，并把这些节点的邻接节点的入度减1，当有新的节点入度更新为0时，再入队列。

如果图存在环，肯定是有节点的入度最终不是0，进不了队列。换句话说，出队列的个数和课程总数相同就可以说明图不存在环。

dfs

按照dfs递归压栈出栈的特性，dfs方法结束的顺序和拓扑排序的顺序是相反的，所以增加了一个栈来储存节点，最后依次出栈到结果集。

bfs

bfs是借助队列实现的，出队列的顺序是和拓扑排序的顺序是一致的，都是入度为0的节点先出队列（相应的，拓扑排序就是入度小的在前面）
